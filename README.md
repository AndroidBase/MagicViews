MagicViews
==========

Provides a simple way to set custom font in Android applications.




## Usage

Add the library as a dependency to your ```build.gradle```

```groovy
debugCompile 'com.ivankocijan.magicviews'
```


#### Version 1.0
* Added support for TextView
